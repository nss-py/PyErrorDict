NameError

NameErrorは文字通りオブジェクトの名前に関するエラーです。未定義の変数や関数をしようとしたときに発生します。

NameError: name 'x' is not defined

print(x)
================================
print(x)
NameError: name 'x' is not defined

単に未定義であった場合は原因がわかりやすいですが、既存の変数・関数のスペルミス、大文字小文字のミスでも同様に未定義として扱われる点は注意が必要です。

point = 70

if point > 70:
  result = "合格"
elif point < 70:
  result = "不合格"

print(result)

print(result)
NameError: name 'result' is not defined

当然ですが、このようにif文中などで定義していて、かつ実行時にその行を読まなかった場合もNameErrorが発生します。上記の例はわかりやすいものですが、複雑な分岐を繰り返したプログラムの際は、それぞれのオブジェクトがきちんと処理されているかを確認しましょう。
