このサイトについて

サイトのデザインはなるべく軽量で見やすいものを目指しています。現在はpython製の静的サイトジェネレータであるpelicanを使い、ここのaboutwilsonというテーマを少々いじったものを使用していますが、後々変更するかもしれません。
