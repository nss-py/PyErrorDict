SyntaxError

SyntaxErrorは日本語に訳すと「構文エラー」のことです。つまり、プログラム中の演算子の忘れや括弧の閉じ忘れなどが原因のエラーです。

SyntaxError: invalid syntax

invalid (無効な) syntax (構文) というエラーです。 

x = 10
y = x + 1

if y = 11:
  print('ok')

================================
if y = 11:
SyntaxError: invalid syntax

x = 10
y = x + 1

if y => 10:
  print('ok')

================================
if y => 10:
SyntaxError: invalid syntax

最も多い原因の一つは、比較演算子でのエラーだと思います。

イコールを2つ続けることを忘れていたり、等号付き不等号を書く際の大なり・小なりとイコールの順序を間違えていることがあるため、コーディングの時は気を付けましょう。

SyntaxError: EOL while scanning string literal

EOLとはEnd Of Fileの略です。文字列が閉じられないままコードがファイルの最後まで読まれたことが原因です。

コード中のクォーテーション・ダブルクォーテーションの閉じ忘れがないかを確認しましょう。

print('Hello, World!)

================================
print('Hello, World!)
SyntaxError: EOL while scanning string literal
