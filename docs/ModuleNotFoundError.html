ModuleNotFoundError

存在しないモジュールをimportしようとすると発生します。

>pip list
Package    Version
---------- -------
pip        21.0.1
setuptools 54.1.2
wheel      0.36.2

import numpy
================================
ModuleNotFoundError: No module named 'numpy'

上の例ではnumpyが存在しない環境でnumpyをimportしようとしています。

pip install numpyなどでインストールしましょう。

インストールしているはずなのにこのエラーが出てくる場合は、pythonやモジュールのパスの確認、モジュールの再インストール、環境変数の確認などをしましょう。

モジュールの再インストールがうまくいかない場合は、pipのキャッシュが原因の可能性もあるためpip uninstall hogehogeの後にpip --no-chache-dir install hogehogeを実行することでうまくいくかもしれません。
